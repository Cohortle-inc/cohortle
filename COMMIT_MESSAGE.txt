feat(mvp-completion-gaps): Add LiveSessionContent and progress calculation system

## Summary
Implemented LiveSessionContent component and comprehensive progress calculation 
system for the MVP completion gaps spec. Added extensive property-based testing 
with optimized iteration counts for faster test execution.

## Features Added

### 1. LiveSessionContent Component
- Session status indicators (upcoming/live/completed)
- Automatic status calculation based on date and duration
- Google Calendar integration with one-click add
- .ics file download for other calendar apps
- Timezone-aware date/time display
- Conditional UI based on session status
- Join button with animated "Live Now" state
- Responsive design with mobile support

### 2. Progress Calculation System
- Core utilities for calculating progress percentages
- Week-level progress calculation
- Programme-level progress aggregation
- Module-level progress tracking
- Real-time progress updates after lesson completion
- React Query hooks for progress management
- Optimistic updates with cache invalidation
- Edge case handling (zero lessons, negative values, etc.)

### 3. React Query Integration
- `useProgrammeProgress()` - Fetch and calculate programme progress
- `useModuleProgress()` - Fetch and calculate module progress
- `useUpdateProgress()` - Update progress cache after completion
- `useInvalidateProgress()` - Bulk refresh of progress queries
- 30-second stale time for frequently changing data
